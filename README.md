# Watch and download on Shikimori
The add-on adds the ability to watch and download anime on the Shikimori.one website.

## Install
### Mozilla Firefox
[![image](https://img.shields.io/amo/v/watch-on-shikimori?color=orange&style=for-the-badge&logo=firefoxbrowser)](https://addons.mozilla.org/en-US/firefox/addon/watch-on-shikimori/)

## Add-on features
- Shikimori-style "Show Player" button, players, and "on other sites" links.
- Kodik and AniLibria player with the ability to select resolution (up to 1080p), playback speed, dubbing, subtitles.
- The player immediately shows the episode where you left off.
- Ability to show and hide the player.
- Direct links to search for the current anime on a torrent tracker.

## How to use?

1. Click the button to show the player.

![image](assets/screenshots/screenshot_1.png?raw=true)

2. Watch anime using the built-in Kodik or AniLibria player with multiple dubbing and subtitle options.

![image](assets/screenshots/screenshot_2.png?raw=true)

3. Click the button again to hide the player.

![image](assets/screenshots/screenshot_3.png?raw=true)

4. Use the provided links to find and download anime from supported torrent trackers.

![image](assets/screenshots/screenshot_4.png?raw=true)
